
# PRecisa de pc e vai apagar seus arquivos

# 1° PASSO 
- # MTK-CLIENT (https://github.com/bkerler/mtkclient)
- # funciona no windows também mas é recomendavel linux

- # para aqueles que não querem ter trabalho de instalar, existe uma opção já feita chamada mtk-client gui 2.0 pesquise na internet e você encontrará

# 2° passo
- # vá nas configurações do seu aparelho e na ultima linha haverá algo como "sobre o dispositivo" clique nele role até a ultima linha vai ter um nome "numero de versão" clique nele 7 vezes até aparecer uma mensagem dizendo que as opções de desenvolvedor foi ativa. depois de ativar volta para as configurações e clique em sistema, avançado e opções de desenvolvedor, dentro dela você vai procurar "desbloqueio de oem" ative ela e agora você poderá seguir para passo 3°

- # ordem: configurações -->sobre o dispositivo ou telefone --> número da versão --> configurações --> sistema --> avançado --> opções de desenvolvedor --> desbloqueio de oem.

 



# 3° passo

- # desligue o aparelho e coloque o mtk client em ativação no windows ou linux.
- # com o aparelho desligado você vai pressionar essa seguinte ordem: botão google ok (fica acima do volume) -> botão de volume (+) e (-). agora você vai segurar todos esses que eu falei mais o botão de ligar e quando você fizer isso rapidamente conecte o cabo no seu celular se tudo ocorrer bem será reconhecido.


# 4° passo 

- # faça um backup das suas partições clicando em select all dump e não faça backup da userdata.





- # agora vai até :
```
https://github.com/R0rt1z2/kaeru/releases
```

- # e escolha o nome do seu dispositivo  no meu caso é Motorola Moto E7 Power (malta_l-kaeru.bin), baixe o .bin

- # agora você para a parte write que fica ao lado da read lá você vai proucurar a lk_a e lk_b. clique em set  e selecione o malta_l-kaeru.bin tanto no lk_a quanto no lk_b. 


- # vá até erase ao lado de write e selecione (metadata,md_udc e userdata) e depois clique em erase.


- # em flash tools clique em unlock bootloader e pronto está desbloqueado.
